## 🚀 Features

### 👤 For Users (Parents)
- View association information and services  
- Book appointments through a multi-step form  
- Provide child information:
  - Age range  
  - Symptoms (multiple choice)  
  - Treatment history  
- Select:
  - Appointment date  
  - Time slot  
- Contact the association  

---

### 🏢 For Association Staff
- Receive booking notifications in Google Sheets  
- View all appointment requests  
- Access structured data:
  - Parent name  
  - Phone number  
  - Appointment date  
  - Child symptoms  

---

## 📝 Booking Form Details

### Step 1: Child Information
- Age range  
- Observed symptoms  
- Treatment history  

### Step 2: Appointment Details
- Parent name  
- Phone number (with validation)  
- Desired appointment date  
- Selected time slot  
- ❌ No booking on weekends (Saturday & Sunday)  

---

## 🏠 Home Page
- Association logo  
- Mission statement  
- Services presentation  
- Call-to-action buttons  

---

## 📊 Database (Google Sheets)
- Acts as a cloud database  
- Stores:
  - Appointments  
  - User data  
- Real-time updates  
- Lightweight and easy to manage  
- Ideal for small to medium projects
